CSS Specificity Cheatsheet

Specificity Tags

Specifier Example Description
Inline Style <h1 style="color: pink;"> highest spec. (other than !important) defined in the HTML itself
element p Applies changes to all p or paragraph elements
class .test Applies changes to all test class elements
ID #navbar Applies changes to elements with the id navbar
Subselector .container h1 Applies changes to h1 elements of the container class
Multiselector h1, h2 Applies changes to all h1 and h2 elements
Universal Selector * Applies changes to all elements
Important Selector !important Overrides all other specificities

Calculating Specificity

Selector Value Calculation
<p style="color: pink;"> 1000 1000 for inline
p 1 1
p.test 11 1 + 10
p #demo 101 1 + 100
#demo 100 100
.test 10 10
p.test1.test2 21 1 + 10 + 10
* 0 0 (universal selector is ignored)
